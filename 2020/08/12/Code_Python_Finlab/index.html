Investing V.S. Gambling

心理上，賭博與投資兩者的目標都是賺錢沒錯。

  • 投資
    是有策略設定一個比較中長期的時間，比如三年、五年或更久，並且計畫累積資產來達到既定的目標
  • 賭博則是帶有比較娛樂及玩票性質，過程中若讓您一時賺到錢而趕到情緒亢奮與高昂，那麼這就比較接近賭博。
    為什麼要投資?
    可以換時間，早一點獲得自由人生。
