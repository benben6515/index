超速學習

發表於 2021-02-24 更新於 2021-03-07 分類於 閱讀筆記

超速學習 | Ulitmate Learning

- Socott H. Young

超速學習是最終生存技能

ISBM: 978-986-175-553-3

推薦序

  • 被動學習帶來知識，主動學習則帶來技能
  • 快速學習是任何人都能擁有的技能
  • 超速學習是高效的自我投資
  • 實現內心的夢想，超速學習是聰明的捷徑
  • 面對消失的工作，超速學習是最強的生存技能

你能不在上MIT的情況下，學會MIT的某套課程嗎?

  • 作者的MIT挑戰
    • 一年上完四年課程
    • 三個月說一口流利外語
    • 一個月學會素描
  • Roger Craig : 什麼問題都考不倒
  • Eric Barone : 從最低薪到百萬富翁
  • 超速學習並不簡單，過程困難又令人沮喪，但能讓你精通目前看似不可能學會的事

超速學習為何重要?

  • 超速學習 : 一種獲取技能與知識的策略，兼具自主性與高強度
  • 該超速學習的理由
    • 經濟因素 : 再見，平庸時代
    • 教育因素 : 學費太高了
    • 科技因素 : 學習的新領域
  • 學習的核心 : 開拓視野，看見之前看不見的事物，認出原本就存在的於自身的能力
  • 超速學習也能活化你已花在學習上的時間

如何成為超速學習者

  • 成為跨領域的超速學習者
  • 克服恐怖谷情緒 : 大量練習
  • 九大法則
  • 決定你想學什麼、怎麼學，並精心策畫出個人計畫

法則一 : 後設學習 - 先畫一個學習地圖

如果說我比別人看得更遠，那是因為我站在巨人的肩膀上。 - 牛頓

  • 後設學習(metalearning) : 學習關於學習這件事
  • 決定為什麼、做什麼、怎麼做
    • 深入研究的好方法 : 專家訪談法
      • 概念 ⇒ 需要了解的事
      • 事實 ⇒ 需要記憶的事
      • 程序 ⇒ 需要練習的事
    • 標竿學習法 : 課程大綱
    • 強調/排除法 : 找出與目標一致的課程
    • 你應該做多少計畫?
      • 10%法則 : 投入預計學習時間的10%了解學習的共同方法、資源、工具與優缺點
      • 報酬邊際效益 : 當你學習的越多越應該繼續投入研究，如果效益越來越少，此時就可以專注學習

法則二 : 專心致志 - 把刀摩利

現在我比較不會分心了。 - (右眼失明的瑞士數學家)里昂哈德.歐拉

  • 無法開始專注(即拖延)
    • 永承認自己正在拖延
    • 花5分鐘去做 ⇒ 番茄鐘
  • 無法持續專注(即分心)
    • 心流&刻意練習的矛盾 : 一次學習15min-1hr，最好涵蓋數個主題
    • 分心來源 : 環境、學習任務、心
  • 無法做到真正的專注
    • 興奮感與工作複雜度有關，最佳化你的興奮度，以維持理想的專注力
  • 承認你所處的狀態，然後從小處著手

法則三 : 直截了當 - 走最短的路，勇往直前

能找到噴泉的人，不會去找水罈。 - 達文西

  • 真正的學習其實在直接做你想學的事才會發生
  • 學會了某領域較深的知識後，會讓未來的學習更具彈性
  • 直接學習的具體方法
    • 以計畫為基礎的學習(寫論文)
    • 沉浸式學習(環境、Open Source)
    • 飛行模擬法(找私人家教)
    • 矯枉過正法(提高挑戰性、目標)
  • 每次學習新事物，詢問自己會在何處及如何表現

法則四 : 反覆操練 - 直擊你最大的弱點

處理好每個小節，整首曲子就會自行完成。 - (作曲家)菲力普.江士頓

  • 找出你學習的速率決定步驟
  • 先直接學習再反覆操練，交替做
  • 操練是辛苦且經常不舒服的
    • 切割時間法
    • 認知練習法
    • 模仿法
    • 放大鏡法
    • 連動法
  • 在心理上感到費力的事，比感到輕鬆的事能為學習帶來更多好處

法則五 : 提取記憶 - 用測驗來學習

等待定努力從內心去回憶，效果會比再看一次書要好。 - (心理學家)威廉.詹姆斯

  • 被動複習時不會得到任何回饋，回饋通常伴隨測驗而來
  • 人類沒有確知自己學得多好的能力
  • 練習的重點 : 找到對的中間點，久到足以記住被提取的記憶，但又不至於忘掉
  • 試著提取尚未存在的答案，就像在餔設一條道路
  • 如何練習提取記憶
    • 快閃卡 : 「暗示 - 回應」框架
    • 自由回想法 : 在白紙上寫下能記下的內容
    • 試題簿法 : 把紀錄改成以後要回答的問題
    • 自創挑戰 : 用實例來說明技巧
    • 闔上書本 : 讓知識都存在腦中

法則六 : 意見回饋 - 別閃避重拳

再被迎面痛擊之前，每個人都有自己的一套計畫。 - 拳王泰森

  • 回饋越多不見得越好，在於回饋的類型
  • 回饋的類型
    • 結果型回饋 : 做錯了嗎?(好/不好)
    • 資訊型回饋 : 哪裡做錯了?(即時回應錯誤)
    • 改正型回饋 : 如何修正錯誤?(老師、教練)
  • 要知道何時該選擇忽略某些內容，以擷取真正有用的資訊
  • 立即回饋能較快認知到錯誤，可以設個計時器，先努力思考
  • 如何改善回饋?
    • 消除噪音(小心隨機性的噪音)
    • 擊中有效的打擊點(避免自我感覺良好或不好)
    • 可據以改變的回饋(學習效率，不同的學習方法)
    • 高強度且快速的回饋(更頻繁得到回饋)

法則七 : 保留記憶 - 別往有洞的桶子倒水

記憶是思考的殘渣。 - (認知心理學家)丹尼爾.威林漢

  • 年資越長的醫生，提供的照顧越差，因為已漸漸忘記知識
  • 遺忘理論
    • 衰退 : 隨著時間遺忘(但兒時的部分記憶仍記得)
    • 干擾 : 新記憶覆蓋舊記憶(順向 : 相似 ； 逆向 : 學習第二外語)
    • 線索 : 沒有鑰匙的盒子(並非遺忘，只是無法取得)
  • 保留記憶機制
    • 間隔法 : 用重複來記憶
    • 程序化 : 習慣性會持續下去
    • 過度學習 : 超越完美的練習
    • 記憶法 : 一張圖可以記一千個字

法則八 : 培養直覺 - 先深掘，再累積

了解一個陳述的意義之前，別問是否正確。 - (數學家)艾瑞特.畢肖普

  • 直覺是大量有條理處理問題的經驗
  • 如何建立直覺
    • 別輕易放棄困難的問題(掙扎計時器)
    • 用證明過程來理解(理解深度的錯覺)
    • 永遠從具體例子開始(我如何思考思訓的)
    • 別欺騙自己(達客效應，問題越學越多)
  • 費曼技巧
    • 在一張紙的最上方，寫下你想理解的(觀念/問題)
    • 在下方解釋那個(觀念/問題)並說明如何(傳達給別人/合理的步驟)
    • 當卡住了，回頭找參考資料，直到找出答案
  • 應用費曼技巧
    • 你以為無法解決的問題
    • 為了擴展直覺

法則九 : 勇於實驗 - 往舒適圈外探索

成果? 為什麼要成果? 我有一堆成果! 其中許多根本行不通。 - 愛迪生

  • 新手與老手的差異，再於高手知道最佳解決方式
  • 實驗的類型
    • 學習資源(大量方法、素材、資源)
    • 技巧(我接下來要學什麼?)
    • 風格的實驗(自己的獨特風格)
  • 成長心態是看見改善的機會與潛力；實驗心態則是制定一個計劃去達到改善的目的
  • 如何成功實驗?
    • 先模仿，再創造
    • 把各種方法放在一起做比較
    • 採用新的限制
    • 混和不相關技能
    • 探索極限
  • 學習本身就是某種嘗試與犯錯
  • 試驗不同方法，找到對你最有效的那一種

你的第一個超速學習計畫

開始，永遠都在今天 - (科學怪人作者)瑪莉.雪萊

  • 步驟一 : 深入研究
    • 你打算學習的主題與大致範圍
    • 你打算使用的主要資源
    • 了解他人如何學會技能的基準點
    • 直接練習活動
    • 備用教材與操練
  • 步驟二 : 安排時間
    • 事先決定投入多少時間
    • 間隔的時間區塊有利於記憶
    • 將大目標拆成較小目標
    • 先測試周行程表
  • 步驟三 : 執行計畫
    • 無論你要執行什麼計畫，現在就是最好的時機
    • 個階段檢視學習的進度
  • 步驟四 : 檢視成果
    • 完成計畫後，花點時間分析，為下一個計劃做準備
  • 步驟五 : 選擇維持或專精所學
    • 選擇維持(投入足夠的時間來保留技能)
    • 重新學習(花一小部分時間足以活化大部分知識)
    • 精通技能(深入你所學過的技能)
  • 超速學習的替代選擇
    • 低強度習慣(維持學習的樂趣)
    • 正式有組織的教育(超速學習不與學校學習矛盾，而是提供更寬廣的路)
  • 終身學習
    超速學習的目標，是要擴展你能得到的機會，並促進使極學習
